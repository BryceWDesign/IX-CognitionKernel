## Project Identity

Repository name: IX-CognitionKernel

Project purpose: a governed causal cognition kernel research project intended to
pursue a disciplined path toward increasingly general, evidence-bound agentic AI
capability through belief tracking, uncertainty exposure, world-model planning,
evaluator-driven learning, memory quarantine, multi-agent critique, human
authority, and outcome-based self-correction.

## No AGI Claim

IX-CognitionKernel does not claim to be artificial general intelligence.

IX-CognitionKernel does not claim to be certified AGI, independently validated
AGI, production-ready autonomy, safety-certified software, security-certified
software, procurement-ready software, or a system suitable for unsupervised
operational decision-making.

The project explicitly reserves any AGI claim for Wave 6 only if overwhelming,
durable, independently validated evidence justifies that claim.
